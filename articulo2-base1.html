TCP

El protocolo TCP (Protocolo de Control de Transmisión) es uno de los protocolos fundamentales en Internet, permitiendo que las aplicaciones se comuniquen con garantías...

FUNCIONAMIENTO

  • TCP tiene un mecanismo complejo de control de errores.

  • Los datos llegan en orden, es decir, en el mismo orden que fueron emitidos.

  • El protocolo TCP permite realizar control de flujo.

  • TCP también dispone de control de congestión.

UDP

El protocolo UDP (Protocolo de Datagramas de Usuario) permite la comunicación sin garantías de fiabilidad, lo que lo hace más rápido que TCP...

FUNCIONAMIENTO

  • Este protocolo no proporciona ningún tipo de control de flujo.
  • No proporciona ningún tipo de control de congestión.
